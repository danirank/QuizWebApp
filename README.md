# Quiz Web App (SPA) – Vanilla JavaScript

En enkel och snabb Quiz-app byggd som en **Single Page Application (SPA)** med **Vanilla JavaScript**, HTML och CSS.  
Appen låter användaren spela quiz, se resultat direkt och hanterar frågor dynamiskt utan sidladdningar.

## Funktioner
- ✅ SPA-flöde (endast index.html)
- ✅ Flera frågor med svarsalternativ
- ✅ Poängräkning och resultatvy
- ✅ Markering av rätt/fel svar
- ✅ Enkel och responsiv layout

## Teknik
- HTML
- Vanilla JavaScript (ingen framework)
- CSS
- Bootsrap
